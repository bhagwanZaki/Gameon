# Gameon

Gameon is an cricket sport ground schduling app.

<h6>
Its an assignment for internship and thus contains only UI part.
</h6>

## Version

Flutter: v3.3.3
Android SDK: v 33.0.0

## Setup

### Clone the Project

```bash
    git clone https://github.com/bhagwanZaki/Gameon.git
```

### Get the packages

```dart
    flutter pub get
```
